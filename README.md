# Netflix- Netflix User Data Analysis
Overview

This project analyses a dataset of Netflix users using Python. The goal is to explore user information such as subscription types, countries, gender distribution, devices used, and age. The analysis includes statistical summaries, data cleaning, filtering, and visualizations.

Technologies Used

Python

NumPy

Pandas

Matplotlib

Installation

Before running the code, install the required libraries:

pip install numpy pandas matplotlib

Dataset

The project uses the dataset:

Netflix_Userbase.csv

Make sure the CSV file is located in the same directory as the Python script.

Features

The analysis includes:

Loading and inspecting the dataset

Checking data structure and missing values

Descriptive statistics

Frequency analysis (country, gender, device, subscription type)

Data filtering by selected countries

Pie charts and bar charts for visualization

Crosstab analysis between subscription type and gender

Pivot tables for multi-variable analysis

Example Analysis

The script generates:

Distribution of users by country

Distribution of subscription types

Gender distribution

Device usage

Statistical summaries of numerical and categorical data

How to Run

Install Python and the required libraries.

Place the dataset in the project folder.

Run the Python script.

python analysis.py
